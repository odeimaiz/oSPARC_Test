# Front-end Prototype using [React.js and plotly](https://github.com/plotly/react-plotly.js/)

## Description
  - The goal of this prototype is to integrate plotly.js in a React application


## Setup

 - Running in dev-mode
``` bash
  cd react-plotly

  sudo make build

  # run container in dev mode
  sudo make run
```
